# 科学计算器的设计与实现

<img src="assets/calculator.gif" align="right" width=350>本项目实现了一个科学计算器，其特性有

- 三角函数，对数函数，指数函数，根号，平方，阶乘【重要：传入的数目前只允许是常数】
- 包含π，自然数e
